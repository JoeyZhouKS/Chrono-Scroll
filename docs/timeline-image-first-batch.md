# 首批事件图片生成清单

下面 12 张用于验证整体风格。全部按 `docs/timeline-image-spec.md` 执行。

## 统一负面要求

```text
No text, no captions, no watermark, no logo, no modern objects, no gore, no cinematic poster typography, no fantasy armor, no neon colors, no photorealistic celebrity faces.
```

## Prompts
